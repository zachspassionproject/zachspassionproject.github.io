Intro to HTML

HTML5 Logo

HTML is one of the most popular programming

languages, mostly used to make websites. HTML

stands for HyperText Markup Language. This entire

website was created using HTML!

Tags

Tags are the main part of code used to tell

the code runner that anything inside the tags

are elements of the page. For example, the paragraph you

just read was made using an h1 tag.

CSS: Cascading Style Sheets

CSS Logo

CSS, or Cascading Style Sheets, is a separate type

of code used to style different parts of HTML.

It is kept in a different file and linked to the HTML

page that it's styling. CSS uses IDs embedded in HTML

tags to know which element is which. CSS takes ALOT

of code though. The file used to style this website

has over 200 lines of code!

Some History

HTML was created by Tim Berners-Lee in 1990 at CERN,

a world-leading laboratory for particle physics

located in Geneva, Switzerland. Tim was trying to

solve the problem of sharing information across

multiple incompatible computer services, and while

he was tinkering with already existing hypertext

technology, he acidentaally created HTML,one of

the most useful code languages in the world and

the very code used to make this website!
